# 盒模型

# 标准盒模型



标准 W3C 盒子模型的范围包括 margin、border、padding、content，并且 content 部分不包含其他部分。

# IE盒模型



IE 盒子模型的范围也包括 margin、border、padding、content，和标准 W3C 盒子模型不同的是：IE 盒子模型的 content 部分包含了 border 和 pading。
